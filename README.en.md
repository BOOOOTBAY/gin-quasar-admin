## Installation：

> Frontend：
>
> ```js
> // cd toyourpath/GQA-FRONTEND：
> yarn
> // run：
> yarn dev
> // build：
> yarn build
> ```

> Backend：
> 
> ```go
> # cd toyourpath/GQA-BACKEND：
> go run main.go
> 
> # build（Linux）：
> set GOARCH=amd64
> set GOOS=linux
> go build main.go
> ```
